Start XR/AR 작동 판정

  1. raw_xyz가 변하면 OpenXR hand input은 들어온다.
  2. filtered_xyz가 변하면 RDF gain/filter는 통과한다.
  3. step_xyz가 변하면 Forge adapter는 통과한다.
  4. eef_delta_norm이 0보다 크면 robot fingertip이 실제로 움직인다.

화면 marker: green=current robot fingertip, cyan=hand delta target, yellow=applied step target, magenta=Forge hole 기준 target.

--no-start-xr 사용 기준

  • ALVR Dashboard와 SteamVR을 이미 PC에서 켜 둔 경우에만 사용한다.
  • Quest 3 안의 ALVR 앱 연결은 자동화되지 않으므로 항상 사용자가 직접 연결한다.
  • 자동 시작을 원하면 명령 끝의 --no-start-xr를 제거한다.
  • 어느 방식이든 Isaac 종료 후에 post snapshot과 진단을 실행한다.
