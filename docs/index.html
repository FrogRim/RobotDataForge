Robot Data Forge Documentation

XR teleoperation trajectory를 학습 후보 dataset artifact로 만드는 문서 허브

Robot Data Forge는 VLA나 World Foundation Model을 직접 만드는 프로젝트가 아니다. RDF는 downstream BC, RL, VLA, WFM 시스템이 학습할 수 있는 replay-verified, action-labelled, task-validated XR teleoperation dataset을 만드는 데이터 인프라다.

현재 proof split

  • MVP-1 PASS learning-ready dataset artifact
  • MVP-2 NEXT learning-proven downstream value
  • CLAIM BOUNDARY policy uplift는 아직 주장하지 않음

주요 보고서

공개 포트폴리오에서 먼저 보여줄 문서들이다. MVP-1은 learning-ready pipeline proof, MVP-2는 learning-proven proof strategy로 분리되어 있다.

MVP-1 pipeline evidence

MVP-1 완료 조건은 policy uplift가 아니라 validated dataset pipeline이다. raw evidence는 보존하고, training eligibility는 별도 gate로 승격한다.

1. Store raw

XR/HMD teleoperation trajectory를 관대하게 저장하고 raw evidence를 보존한다.

2. Evaluate

task_state, task outcome, data quality, operator/evaluator success를 분리한다.

3. Gate

replay/action contract와 curation reason을 기록하고 accepted/rejected manifest를 만든다.

4. Export

HDF5 export와 trainer smoke를 통과한 learning-ready dataset artifact로 만든다.

추가될 문서 흐름

앞으로 문서가 늘어나면 이 허브에서 보고서, 기술 명세, 실험 결과, 공개 글감을 분리해서 연결한다. MVP-2 결과는 positive/negative 여부와 관계없이 learning-proven evidence로 별도 축적한다.
